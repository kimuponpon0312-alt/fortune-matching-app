# 運命のマッチング - 四柱推命マッチングアプリ

30〜50代向けの占いマッチングアプリのプロトタイプです。生年月日から四柱推命の日干を計算し、相性抜群の相手タイプを表示します。

## 機能

- 生年月日から四柱推命の日干（十干）を自動計算
- 親しみやすい名称でタイプを表示（太陽、大樹、宝石など）
- 干合に基づく相性抜群の相手タイプを表示

## 技術スタック

- **言語**: TypeScript
- **フレームワーク**: Next.js 14 (App Router)
- **スタイリング**: Tailwind CSS
- **デザイン**: ダークネイビーとゴールドを基調とした高級感あるデザイン

## セットアップ方法

### 1. 依存関係のインストール

```bash
npm install
```

### 2. 開発サーバーの起動

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてアプリを確認できます。

### 3. ビルド（本番用）

```bash
npm run build
npm start
```

## 使い方

1. 生年月日（年、月、日）を入力
2. 「運命を占う」ボタンをクリック
3. あなたのタイプと相性抜群の相手タイプが表示されます

## 注意事項

- このプロトタイプでは、四柱推命の簡易計算を使用しています
- 正確な四柱推命の計算には旧暦への変換が必要ですが、プロトタイプでは1900年1月1日を基準とした簡易計算を実装しています
